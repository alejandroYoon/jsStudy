총 몇문제인지, 그리고 풀이에 나설 사람들의 이름을 각각 상수와 객체로 입력받아
codingTest 함수의 인자로 넘겨줍니다.

문제는 가장 먼저 5문제를 푼 사람이 우승자가 됩니다.

반환 값은 객체로 가장 많은 코딩 테스트 문제를 풀이한 사람과
각 사람들의 풀이한 문제 수를 출력해줍니다.

input data

const for_number = 60;
const user_name = { 코난: 0, 김전일: 0, 뽀로로: 0, 크롱: 0, 루피: 0, 타요: 0 };

output data

[
    '가장 많은 코딩 테스트 문제를 풀이한 사람은 크롱입니다.',
    { '코난': 3, '김전일': 4, '뽀로로': 2, '크롱': 5, '루피': 3, '타요': 2 }
]

Hint

빌트인 객체 Math의 random 메소드를 사용하면 0 ~ 1 까지의 랜덤값을 생성할 수 있습니다.
빌트인 객체 Math의 round, ceil, floor를 사용하면 각각 반올림, 올림, 내림을 사용할 수 있습니다.
빌트인 객체 Object의 keys를 사용하면 해당 객체의 키값들을 배열 형태로 받아올 수 있습니다.
